# 기억, 사랑 그리고 선택...

지하 한가운데 존재하는 알 수 없는 탑, 그리고 이름.

어느 날 발견한 지도가 약속한 아름다운 삶을 이루기 위해

탑에 오른 소년 조이는 소녀 하피와 함께 서서히 진실과 마주한다.

# 세계관

갈색 빛이 감도는 지하 세계.

이곳에는 지상이 존재하지 않으며, 공간은 끝없이 이어진다.

각자 다른 개성을 지닌 검은 스틱맨들이 이 세계에서 살아가고 있다.

지하 정중앙에는 하나의 국가만 한 거대한 탑이 존재하고,

사람들은 탑에서 흘러나오는 알 수 없는 물질로 삶을 영위한다.

탑의 정체는 지옥일까, 아니면 천국일까…

# 조작키

- 좌우 방향키 : 이동
- 스페이스바 : 점프
- 좌우 방향키 + 아래 방향키 : 구르기
- 좌Shift : 대시
- A키 : 공격 (2단 공격 가능)
- 위 방향키 : 가드
- I키 : 인벤토리
- M키 : 지도
- Esc키 : 설정

# 게임 목표

몬스터를 처치해 레벨업과 아이템을 획득하여 성장하자.
계속해서 성장하여 마을의 퀘스트를 깨기, 특정 맵에 도달, 보스 클리어
등등은 스토리 진행해 끝까지 도달하자.

# 게임 요소

- 레벨 : 올리면 올릴 수록 캐릭터가 점점 강해진다.
- 아이템 : 장착하면 아이템의 특성에 따라 캐릭터의 능력치 상승
- 몬스터 / 보스 : 플레이어에게 공격하는 대상
- 스테미나 : 스킬 사용시 필요한 기력
- 스킬 : 강력한 공격 및 효과. 공격력과 효과는 그 스킬의 필요 스테미나의 양에 의존한다.
- 재화 : 강력한 아이템을 사기위한 돈
- 상점 : 아이템을 사용하는 곳
- 지도 : 맵 이동시 사용 특정 맵 도달이나 이벤트 진행 시 잠금 헤제
- 인벤토리 : 아이템을 담는 곳
- 스토리 : 플레이어가 특정 조건을 달성할 시 나오는 씬
